# India Last Minute Engineering College - Website Specification

## Project Overview

- **Project Name**: India Last Minute Engineering College Website
- **Project Type**: Multi-page static website with form validation
- **Core Functionality**: Academic institute website showcasing departments, courses, faculty, admissions with an online enquiry form
- **Target Users**: Prospective students, parents, and general public

## Pages Structure

### 1. Home Page (index.html)

- Hero section with college name and tagline
- Quick introduction about the college
- Featured sections (Departments, Courses, Admissions highlights)
- Footer with contact information

### 2. Departments Page (departments.html)

- List of engineering departments
- Each department with description and facilities

### 3. Courses Page (courses.html)

- Undergraduate and postgraduate courses
- Course duration, eligibility details

### 4. Faculty Page (faculty.html)

- Faculty members with designations
- Specializations and qualifications

### 5. Admissions Page (admissions.html)

- Admission process timeline
- Eligibility criteria
- Fee structure overview

### 6. Enquiry Form Page (enquiry.html)

- Online enquiry form with validation
- Fields: Name, Email, Phone, Course Interest, Message

## Form Validation Rules (JavaScript)

- **Name**: Required, minimum 3 characters, letters and spaces only
- **Email**: Required, valid email format
- **Phone**: Required, valid 10-digit Indian mobile number
- **Course Interest**: Required, must select an option
- **Message**: Required, minimum 10 characters

## UI/UX Specification

### Color Palette

- Primary: #1a237e (Deep Indigo - represents education/trust)
- Secondary: #f57c00 (Orange - represents energy/modernity)
- Accent: #00bfa5 (Teal - represents growth)
- Background: #f5f5f5 (Light gray)
- Text: #212121 (Dark gray)
- White: #ffffff

### Typography

- Headings: 'Poppins', sans-serif (bold, modern)
- Body: 'Open Sans', sans-serif (readable)

### Layout

- Responsive design
- Navigation bar on all pages
- Consistent header and footer
- Mobile-friendly hamburger menu

### Components

- Navigation bar with links to all pages
- Hero sections
- Cards for departments, courses, faculty
- Form with validation feedback
- Footer with college info
